## Perforce P4 MCP: Natural Language Version Control for Game Developers

Perforce has unveiled **P4 MCP (Model Context Protocol)** at GDC 2026, enabling developers to interact with their game assets and code using natural language. This breakthrough allows teams to query assets, revision history, and dependencies directly within AI coding tools like Claude Code, Cursor, and JetBrains Rider.

The P4 MCP Server maintains strict enterprise-grade permissions and audit trails, ensuring AI-assisted operations remain secure and compliant with existing access controls. This is particularly valuable for AAA studios managing complex asset pipelines with thousands of files.

**Key Features:**
- Natural language queries for asset management
- Integration with popular AI coding assistants
- Maintains existing permission models
- Enterprise-grade security and auditing
