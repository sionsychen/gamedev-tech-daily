## Unity AI Tools Ecosystem 2026

Unity's AI ecosystem continues to evolve with **Unity Muse** and **Unity Sentis** leading the charge. Muse provides an AI assistant suite supporting natural language API queries, code completion, and texture/2D/3D asset generation. Sentis enables running AI models locally during gameplay for NPC behavior decisions and procedural content generation.

**Key Features:**
- Natural language code generation
- Runtime AI inference capabilities
- Integrated asset generation workflow
