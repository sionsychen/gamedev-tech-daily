## 本地预览

```bash
# 安装依赖
gem install bundler jekyll

# 进入项目目录
cd gamedev-tech-daily

# 安装依赖包
bundle install

# 本地运行
bundle exec jekyll serve

# 访问 http://localhost:4000/gamedev-tech-daily
```

## 自动发布流程

1. **每天 11:00** (GMT+8) 自动触发
2. **搜索**最新的游戏技术策划相关文章
3. **筛选**高质量内容（排除视频/低质量来源）
4. **生成** Jekyll 格式的 Markdown 文章
5. **推送**到 GitHub
6. **自动部署**到 GitHub Pages
